# fukui-station-kanko-reservation
こちらが日本語のREADMEです: [README.ja.md](README.ja.md)

このプロジェクトは、福井県福井市の福井駅周辺のホテル予約と稼働率のデータを視覚化するオープンソースのWebアプリケーションを提供しています。

## デモ
https://code4fukui.github.io/fukui-station-kanko-reservation/

## 機能
- 福井駅周辺のホテル稼働率、平均室料(ADR)、RevPAR、およびその他の主要指標を表示します
- ユーザーが選択した期間のデータを表示できます
- CSVフォーマットでオープンデータのダウンロードを提供します

## データ / API
プロジェクトは福井県観光連盟の公開データを使用し、非公開のAPIを通じてアクセスしています。
