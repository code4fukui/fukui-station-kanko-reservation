# 福井駅前エリア 宿泊予約状況

福井県観光連盟が提供する、福井駅前エリアの宿泊施設の予約状況を可視化したオープンデータリポジトリです。

## デモ
https://code4fukui.github.io/fukui-station-kanko-reservation/

## 機能
- 福井駅前エリアの宿泊施設の予約状況を、客室稼働率(OCC)、客室平均単価(ADR)、RevPAR等の指標で可視化
- 期間を指定して過去の予約状況を確認可能
- 宿泊施設の総客室数を表示

## データ・API
- 福井県観光連盟のFTAS(観光データ分析システム)からの予約データ
- 宿泊施設の客室数情報
